Pocketbook apps manager

Esta aplicación es un gestor de aplicaciones no oficial para lectores electrónicos PocketBook. Permite descargar aplicaciones no oficiales para PocketBook.

Instalación

Para instalar la aplicación, conecta tu lector electrónico a tu ordenador mediante un cable y copia el archivo app-manager.app desde release.zip a la carpeta de aplicaciones. En Linux, puedes construir la aplicación a partir del código fuente usando el comando `make build` e instalarla automáticamente en el lector electrónico conectado usando el comando `make install`.

Código fuente y contribución

Puedes obtener el código fuente del gestor de aplicaciones en github. Puedes añadir tu propia aplicación PB. Para ello necesitas copiar Makefile y .github/workflows/uploadOnRelease.yml en tu repositorio y crear una versión. A continuación, necesitas clonar el repositorio esto repo, añadir tu aplicación a repo.json y enviar una solicitud de extracción. Después de que se acepte la solicitud de extracción, tu aplicación podrá ser descargada por el gestor de aplicaciones. También puedes usar el repositorio de plantillas.

Aviso legal

Esta aplicación se distribuye bajo la licencia MIT y se proporciona "tal cual". Las aplicaciones instaladas a través de este instalador también son no oficiales y no vienen con ninguna garantía.

Advertencia

Ten en cuenta que la instalación y el uso de aplicaciones no oficiales en tu lector electrónico PocketBook pueden entrañar riesgos para la estabilidad y la seguridad de tu dispositivo. Las aplicaciones no oficiales pueden no haber sido probadas o validadas a fondo y pueden causar problemas con el rendimiento, los datos o la configuración de tu dispositivo. Antes de instalar cualquier aplicación, asegúrate de comprender los riesgos potenciales y de tomar las precauciones necesarias para proteger tu dispositivo y tus datos.

Preguntas frecuentes (FAQ)

¿Cómo instalo el instalador de aplicaciones en PocketBook?

Para instalar la aplicación, conecta tu lector electrónico a tu ordenador mediante un cable y copia el archivo de lanzamiento en la carpeta de aplicaciones. En Linux, puedes construir la aplicación a partir del código fuente usando el comando `make build` e instalarla automáticamente en el lector electrónico conectado usando el comando `make install`.

¿Cuáles son los riesgos de usar aplicaciones no oficiales?

El uso de aplicaciones no oficiales puede entrañar riesgos para tu lector electrónico, incluyendo la eliminación de archivos o configuraciones importantes del sistema, la modificación de la configuración del sistema, la instalación de software no verificado o malicioso y el acceso a información o configuraciones sensibles.

¿Cómo actualizo el instalador de aplicaciones a la última versión?

El instalador de aplicaciones admite la autoactualización. Si la autoactualización tiene éxito, tendrás la última versión del instalador de aplicaciones. Si la autoactualización falla, puedes actualizar manualmente el instalador de aplicaciones conectando tu lector electrónico a tu ordenador mediante un cable y copiando el nuevo archivo de lanzamiento en la carpeta de aplicaciones, reemplazando el archivo existente.

¿Instalé la aplicación, pero no aparece en la lista de aplicaciones?

En algunos casos, el Pocketbook no actualiza la lista de aplicaciones automáticamente, en cuyo caso reiniciar el libro electrónico debería ayudar.
