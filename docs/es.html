Administrador de aplicaciones para Pocketbook

Esta aplicación es un administrador no oficial de aplicaciones para lectores electrónicos PocketBook. Le permite descargar aplicaciones no oficiales para Pocketbook.

Instalación

Para instalar la aplicación, conecte su lector electrónico a su computadora mediante un cable y copie el archivo app-manager.app desde release.zip a la carpeta de aplicaciones. En Linux, puede construir la aplicación desde el código fuente utilizando el comando make build e instalarla automáticamente en el lector electrónico conectado utilizando el comando make install.

Código fuente y contribución

Puede obtener el código fuente del administrador de aplicaciones en github. Puede agregar su propia aplicación pb. Para hacer esto, necesita copiar Makefile y .github/workflows/uploadOnRelease.yml en su repositorio y crear una versión. A continuación, necesita clonar este repositorio, agregar su aplicación a repo.json y hacer una solicitud de extracción. Una vez aceptada la solicitud de extracción, su aplicación se puede descargar a través del administrador de aplicaciones.

Advertencia

Tenga en cuenta que la instalación y el uso de aplicaciones no oficiales en su lector electrónico PocketBook pueden plantear riesgos para la estabilidad y seguridad de su dispositivo. Las aplicaciones no oficiales pueden no haber sido probadas o validadas exhaustivamente, y pueden causar problemas con el rendimiento, los datos o la configuración de su dispositivo. Antes de instalar cualquier aplicación, asegúrese de comprender los riesgos potenciales y tome las precauciones necesarias para proteger su dispositivo y sus datos.

Preguntas frecuentes (FAQ)

P: ¿Cómo instalar el instalador de aplicaciones en PocketBook?

R: Para instalar la aplicación, conecte su lector electrónico a su computadora mediante un cable y copie el archivo de la versión en la carpeta de aplicaciones. En Linux, puede construir la aplicación desde el código fuente utilizando el comando make build e instalarla automáticamente en el lector electrónico conectado utilizando el comando make install.

P: ¿Cuáles son los riesgos de usar aplicaciones no oficiales?

R: El uso de aplicaciones no oficiales puede plantear riesgos para su lector electrónico, incluyendo la eliminación de archivos o configuraciones importantes del sistema, cambio de la configuración del sistema, instalación de software no verificado o malicioso, y acceso a información o configuraciones confidenciales.

P: ¿Cómo actualizar el instalador de aplicaciones a la última versión?

R: El instalador de aplicaciones admite la autoactualización. Si la autoactualización es exitosa, tendrá la última versión del instalador de aplicaciones. Si la autoactualización falla, puede actualizar el instalador de aplicaciones manualmente conectando su lector electrónico a su computadora mediante un cable y copiando el nuevo archivo de la versión en el carpeta de aplicaciones, reemplazando el archivo existente.

P: ¿Instalé la aplicación pero no apareció en la lista de aplicaciones?

R: En algunos casos, el pocketbook no actualiza la lista de aplicaciones automáticamente, en cuyo caso reiniciar el libro electrónico debería ayudar.
