Pocketbook apps manager

Cette application est un gestionnaire d'applications non officiel pour les liseuses PocketBook. Elle permet de télécharger des applications non officielles pour PocketBook

Installation

Pour installer l'application, connectez votre liseuse à votre ordinateur via un câble et copiez l'application app-manager.app à partir de release.zip dans le dossier des applications. Sur Linux, vous pouvez compiler l'application à partir du code source en utilisant la commande make build et l'installer automatiquement sur la liseuse connectée en utilisant la commande make install.

Code source et contribution

Vous pouvez obtenir le code source du gestionnaire d'applications sur github;. Vous pouvez ajouter votre propre application PB. Pour cela, vous devez copier Makefile et .github/workflows/uploadOnRelease.yml dans votre dépôt et créer une version. Ensuite, vous devez cloner ce dépôt, ajouter votre application à repo.json et faire une demande de pull. Après acceptation de la demande de pull, votre application peut être téléchargée par le gestionnaire d'applications

Avertissement de non-responsabilité

Cette application est licenciée sous la licence MIT et est fournie "telle quelle". Les applications installées via cet installateur sont également non officielles et ne comportent aucune garantie.

Avertissement

Veuillez être conscient que l'installation et l'utilisation d'applications non officielles sur votre liseuse PocketBook peuvent présenter des risques pour la stabilité et la sécurité de votre appareil. Les applications non officielles peuvent ne pas avoir été suffisamment testées ou validées et peuvent potentiellement causer des problèmes avec les performances, les données ou les paramètres de votre appareil. Avant d'installer toute application, assurez-vous de comprendre les risques potentiels et prenez les précautions nécessaires pour protéger votre appareil et vos données.

FAQ

Q : Comment installer l'installateur d'applications sur PocketBook ?

R : Pour installer l'application, connectez votre liseuse à votre ordinateur via un câble et copiez le fichier de release dans le dossier des applications. Sur Linux, vous pouvez compiler l'application à partir du code source en utilisant la commande make build et l'installer automatiquement sur la liseuse connectée en utilisant la commande make install.

Q : Quels sont les risques de l'utilisation d'applications non officielles ?

R : L'utilisation d'applications non officielles peut présenter des risques pour votre liseuse, notamment la suppression de fichiers système importants ou de paramètres, la modification de la configuration du système, l'installation de logiciels non vérifiés ou malveillants, et l'accès à des informations ou paramètres sensibles.

Q : Comment mettre à jour l'installateur d'applications vers la dernière version ?

R : L'installateur d'applications prend en charge la mise à jour automatique. Si la mise à jour automatique est réussie, vous aurez la dernière version de l'installateur d'applications. Si la mise à jour automatique échoue, vous pouvez mettre à jour l'installateur d'applications manuellement en connectant votre liseuse à votre ordinateur via un câble et en copiant le nouveau fichier de release dans le dossier des applications, en remplaçant le fichier existant.

Q : J'ai installé l'application, mais elle n'apparaît pas dans la liste des applications ?

R : Dans certains cas, la liseuse PocketBook ne met pas à jour automatiquement la liste des applications, auquel cas redémarrer la liseuse devrait aider.
