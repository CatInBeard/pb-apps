Pocketbook apps manager

Cette application est un gestionnaire d'applications non officiel pour les liseuses PocketBook. Elle vous permet de télécharger des applications PocketBook non officielles.

Installation

Pour installer l'application, connectez votre liseuse à votre ordinateur via un câble et copiez le fichier app-manager.app depuis release.zip à le dossier applications. Sous Linux, vous pouvez compiler l'application à partir du code source en utilisant la commande `make build` et l'installer automatiquement sur la liseuse connectée en utilisant la commande `make install`.

Code source et contribution

Vous pouvez obtenir le code source du gestionnaire d'applications sur github. Vous pouvez ajouter votre propre application PB. Pour ce faire, vous devez copier Makefile et .github/workflows/uploadOnRelease.yml dans votre dépôt et créer une version. Ensuite, vous devez cloner le dépôt ce repo, ajoutez votre application à repo.json et envoyez une demande de tirage. Une fois la demande de tirage acceptée, votre application pourra être téléchargée par le gestionnaire d'applications.

Avertissement

Avertissement

Veuillez noter que l'installation et l'utilisation d'applications non officielles sur votre liseuse PocketBook peuvent présenter des risques pour la stabilité et la sécurité de votre appareil. Les applications non officielles peuvent ne pas être soigneusement testées ou validées et peuvent potentiellement causer des problèmes avec les performances, les données ou les paramètres de votre appareil. Avant d'installer des applications, assurez-vous de bien comprendre les risques potentiels et de prendre les précautions nécessaires pour protéger votre appareil et vos données.

FAQ

Q : Comment installer le programme d'installation d'applications sur PocketBook ?

R : Pour installer l'application, connectez votre liseuse à votre ordinateur à l'aide d'un câble et copiez le fichier de publication dans le dossier des applications. Sous Linux, vous pouvez compiler l'application à partir du code source en utilisant la commande `make build` et l'installer automatiquement sur la liseuse connectée en utilisant la commande `make install`.

Q : Quels sont les risques liés à l'utilisation d'applications non officielles ?

R : L'utilisation d'applications non officielles peut entraîner des risques pour votre liseuse, notamment la suppression de fichiers ou de paramètres système importants, la modification de la configuration du système, l'installation de logiciels non vérifiés ou malveillants et l'accès à des informations ou des paramètres sensibles.

Q : Comment mettre à jour le programme d'installation d'applications vers la dernière version ?

R : Le programme d'installation d'applications prend en charge l'auto-mise à jour. Si l'auto-mise à jour réussit, vous disposerez de la dernière version du programme d'installation d'applications. Si l'auto-mise à jour échoue, vous pouvez mettre à jour manuellement le programme d'installation d'applications en connectant votre liseuse à votre ordinateur à l'aide d'un câble et en copiant le nouveau fichier de publication dans le dossier des applications, en remplaçant le fichier existant.

Q : J'ai installé l'application, mais elle n'apparaît pas dans la liste des applications ?

R : Dans certains cas, le pocketbook ne met pas à jour la liste des applications automatiquement, auquel cas le redémarrage du livre électronique devrait aider.
